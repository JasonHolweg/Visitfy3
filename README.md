# 🌐 Visitfy Website

Modern landing page for showcasing immersive **360° virtual tours** for businesses.

---

## 🚀 Overview

Visitfy helps businesses increase visibility, build trust, and generate more customer inquiries through high-quality 360° virtual tours.

This repository contains the official website used to present Visitfy's services, case studies, and contact options in a clear and conversion-focused way.

---

## ✨ Features

* Responsive design (desktop, tablet, mobile)
* Clean and modern UI/UX
* Strong call-to-action sections for lead generation
* Service explanation and benefits
* Case studies and real-world examples
* Contact form with inquiry handling
* Performance-optimized frontend

---

## 🧩 Sections

The website includes:

* Hero section with clear value proposition
* Benefits of 360° virtual tours
* Comparison (Photos vs. 360° experience)
* Live demo examples
* Case studies with measurable results
* Testimonials / partner feedback
* Process explanation (3-step workflow)
* Contact & inquiry form
* FAQ section

---

## 🛠️ Tech Stack

* **PHP** – Server-side rendering, form handling, JSON-based CMS
* **HTML5 / CSS3** – Custom styling, modern layout (Flexbox/Grid), animations
* **JavaScript (vanilla)** – Intro animation, scroll effects, rotating text, cookie consent
* **GSAP + ScrollTrigger** – Smooth scroll-driven animations
* **Mailgun API** – Transactional email delivery for contact forms
* **Cloudflare Turnstile** – Bot protection on forms
* **JSON** – Flat-file content storage (no database)

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/visitfy-website.git
```

2. Navigate into the project folder:

```bash
cd visitfy-website
```

3. Open `index.php` in your browser or run a local server:

```bash
php -S localhost:8000
```
